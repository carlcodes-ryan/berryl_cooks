# Berryl's Global Kitchen

A modern, static recipe website featuring 30 authentic dishes from around the world.

## Deployment

This project is configured for automatic deployment on [Render](https://render.com) using the `render.yaml` Blueprint.

### How to Deploy

1. Push this code to a GitHub or GitLab repository.
2. Log in to your Render dashboard.
3. Select **New +** -> **Blueprint**.
4. Connect your repository.
5. Render will automatically detect the configuration and deploy the site.

### Configuration Details

The `render.yaml` file ensures the site is served correctly, mapping the root URL to `berryl_recipes.html`.
